该项目是基于Unity开发的缝合LOL的斗魂竞技场和海克斯大乱斗+无尽狂潮的类幸存者游戏

核心玩法：装备、海克斯和英雄技能之间的构筑和配合

程序框架：采用RVO2+NavMesh混合寻路，采用Property处理数据，数据变化时调用事件，ViewModel中使用Binder的方法绑定数据与显示（MVVM模式）；获取数据时调用依赖属性，查看是否有脏标记，决定是否更新数据（脏标记模式）。程序设计框架如图所示

<img src="Designs\ProgrammeDesignDiagram.png" alt="程序设计图" style="zoom:50%;" />

数据设计：基础资源有两种，金币和经验值。经验值决定等级，等级再决定基础属性和技能等级，同时根据等级提供海克斯。金币提供购买道具，包括装备和属性锻造器。属性锻造器（后续简称锻体）可以获得三选一的各项基础属性，若不买任何装备锻体10层，则会有概率出增幅碎片（20~80%）增幅所有锻体属性，以拓宽玩法。装备则提供基础属性+特殊技能（有的装备会附带百分比加成）。

<img src="Designs\AttributesDataFlowDiagram.png" alt="数据设计图">

项目采用配置表来配置数据，以方便随时更新。使用python脚本自动将xlsx配置表转化为json文件，以方便游戏开始时动态读取。
